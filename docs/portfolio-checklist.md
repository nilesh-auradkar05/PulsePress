# PulsePress Portfolio Checklist

Phase 1 is portfolio-ready only when:

- Cloud deployment works.
- Acceptance flow in `docs/test-plan.md` passes.
- README explains architecture, local dev, cloud deploy, and failure recovery.
- Architecture diagram exists.
- OpenTelemetry/CloudWatch screenshots exist.
- Load-test report exists.
- Admin recovery screenshot exists.
- Ledger/bill breakdown screenshot exists.
- SSE dashboard screenshot exists.
- Fresh-clone setup succeeds.
- Resume bullets are drafted from verified outcomes only.
